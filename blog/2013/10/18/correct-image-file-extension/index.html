2. 更正文件扩展名

需要用到一些文件操作的方法，用到 os 这个模块，记录一下：

  • os.chdir(dirname): 用于改变工作路径到 dirname。
  • os.listdir(dirname): 列出 dirname 下的目录和文件。
  • os.path.isfile(name): 判断 name 是否为文件。
  • os.path.splitext(): 分离文件名与扩展名。
  • os.rename(a, b): 重命名文件。
